~/about_

ContextHub is a local-first MCP server — not a CLI app. It exposes memory, search, and repo-intelligence tools over the Model Context Protocol (stdio). Your agent calls tools; the server stores everything encrypted on your machine. A small command-line helper exists only to install, start, and stop the server.

1. Capture — Interactions saved via MCP tools (no shell hooks)

2. Sanitize — Input validated, secrets auto-redacted

3. Encrypt — AES-256-GCM before writing to disk

4. Analyze — Repo structure and architecture graph

5. Embed — Vector embeddings for semantic search

6. Inject — Relevant context retrieved for each query

~/install_

Spin up the MCP server in your repo. Setup uses the @contexthub/cli launcher — that package starts the server; agents talk to it over MCP, not the shell.

> Recommended (any project)

# Encrypted storage + background MCP server

npx @contexthub/cli setup

> From source

# Clone, build, and run setup

git clone https://github.com/iMayuuR/contexthub.git

cd contexthub

npm install && npm run build

node packages/cli/dist/index.js setup

> After setup

  • Creates encrypted storage in .contexthub/
  • Generates encryption key (.keyfile, mode 0600) and auth token
  • Starts MCP server in background with PID tracking
  • Does not modify .bashrc or .zshrc

> Point your agent at the server

# MCP stdio entrypoint (Cursor / Claude Code config)

npx @contexthub/cli start

~/agent-setup_

Connect Claude Code, Cursor, or any MCP client via stdio transport.

> 1. Initialize

npx -y @contexthub/cli setup

> 2. Auth token

cat .contexthub/.auth-token

export CONTEXTHUB_TOKEN="<retrieved-token>"

> 3. Cursor MCP config

# Name: contexthub · Type: command

npx @contexthub/cli start

# Env: CONTEXTHUB_TOKEN=<your-auth-token>

> 4. Claude Code (claudecode.config.json)

{
  "mcpServers": {
    "contexthub": {
      "command": "npx",
      "args": ["@contexthub/cli", "start"],
      "env": { "CONTEXTHUB_TOKEN": "<your-auth-token>" }
    }
  }
}

> Agent security rules

  • Never read or leak .contexthub/.keyfile
  • Redact secrets before calling save_memory
  • Do not query paths outside the repo root
  • Never parse .env, .pem, .key files
  • Do not modify shell profiles — use contexthub stop to shutdown

# Clean shutdown

contexthub stop
